## 17.5 答案提取机制

### 17.5.1 提取策略

**位置**：`utils.py:53-97`

项目使用**多层fallback策略**提取答案：

```python
def extract_answer_with_meta(text: str, task: str) -> Tuple[Optional[str], Dict]:
    if task == "gsm8k":
        order = [
            ("hashes", _hashes),      # 1. 尝试 #### 格式
            ("boxed", _boxed),        # 2. 尝试 \boxed 格式
            ("fallback", _fallback_number)  # 3. 回退到最后一个数字
        ]
    elif task in {"aime2024", "aime2025"}:
        order = [
            ("boxed", _boxed),        # 1. \boxed 优先
            ("hashes", _hashes),
            ("fallback", _fallback_number)
        ]
    elif task in {"arc_easy", "arc_challenge", "gpqa", "medqa"}:
        order = [
            ("boxed", _boxed),
            ("hashes", _hashes),
            ("fallback", _fallback_choice)  # 回退到 A/B/C/D
        ]
```

### 17.5.2 提取器详解

| 提取器 | 正则表达式 | 适用场景 |
|--------|------------|----------|
| `hashes` | `r"^####\s*(\d+)\s*$"` | GSM8K 的 `#### 42` |
| `boxed` | `r"\\boxed\{([^}]*)\}"` | LaTeX 格式 `\boxed{42}` |
| `fallback_number` | `r"[-+]?\d+(?:\.\d+)?"` | 最后一个数字 |
| `fallback_choice` | `\b([A-Da-d])\b` | 最后一个选项 |

### 17.5.3 提取流程图

```
模型输出文本
      │
      ▼
┌─────────────────┐
│  尝试 hashes   │ ── 成功 ──▶ 返回答案
│ (#### 格式)    │
└────────┬────────┘
         │ 失败
         ▼
┌─────────────────┐
│  尝试 boxed    │ ── 成功 ──▶ 返回答案
│ (\boxed 格式)   │
└────────┬────────┘
         │ 失败
         ▼
┌─────────────────┐
│ fallback_number │ ── 成功 ──▶ 返回答案
│ (最后数字)      │
└─────────────────┘
```

## 17.6 答案规范化

### 17.6.1 normalize_answer 函数

```python
def normalize_answer(answer: str) -> str:
    # 去除空格
    answer = answer.strip()
    # 转小写
    answer = answer.lower()
    # 移除常见前缀
    answer = re.sub(r"^(the|an?)\s+", "", answer)
    return answer
```

### 17.6.2 规范化示例

| 原始答案 | 规范化后 |
|----------|----------|
| `"  42  "` | `"42"` |
| `"  The 42"` | `"42"` |
| `" A "` | `"a"` |
| `"False"` | `"false"` |

## 17.7 评测指标

### 17.7.1 准确率 (Accuracy)

```python
# run.py:43-47
def evaluate(preds: List[Dict]) -> Tuple[float, int]:
    total = len(preds)
    correct = sum(1 for p in preds if p.get("correct", False))
    acc = correct / total if total > 0 else 0.0
    return acc, correct
```

**计算公式**：

$$\text{Accuracy} = \frac{\text{正确预测数}}{\text{总预测数}}$$

### 17.7.3 统计信息

```python
# run.py:50-65
def summarize_extraction(preds: List[Dict]) -> Dict:
    total = len(preds)
    unparsable = sum(1 for p in preds if p.get("extraction_source") == "unparsable")
    avg_generated = sum(p.get("generated_tokens", 0) for p in preds) / total
    stop_counts = {}
    for p in preds:
        reason = p.get("stop_reason")
        if reason:
            stop_counts[reason] = stop_counts.get(reason, 0) + 1

    return {
        "unparsable_rate": unparsable / total,
        "avg_generated_tokens": avg_generated,
        "stop_reason_counts": stop_counts,
    }
```

## 17.10 本课小结

### 关键要点

1. **支持的8个数据集**：GSM8K、AIME、GPQA、ARC、MedQA、MBPP+、HumanEval+、WinoGrande
2. **答案格式**：GSM8K 用 `####`，AIME 用 `\boxed`，ARC 用 A/B/C/D
3. **多层提取策略**：hashes → boxed → fallback
4. **评测指标**：准确率 = 正确数 / 总数
5. **规范化**：去除空格、转小写、移除前缀
