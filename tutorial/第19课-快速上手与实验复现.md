## 19.3 快速实验：单模型训练

### 19.3.1 准备锚点文本

如果还没有锚点文本，使用默认的：

```bash
# 默认会使用代码中的 10 条默认锚点文本
# 也可以准备自定义锚点文本
echo '["1+1=2", "2+2=4", "猫是动物"]' > data/anchor.jsonl
```

### 19.3.2 训练命令

```bash
python train_vision_latent_mas_codec_new.py \
  --model_name "Qwen/Qwen3-VL-2B-Thinking" \
  --vision_codec_path checkpoints/codec_qwen3vl2b.pt \
  --vision_codec_anchor_texts_path data/anchor.jsonl \
  --vision_codec_dim 512 \
  --vision_codec_tokens 1024 \
  --vision_codec_img_tokens 256 \
  --vision_codec_heads 8 \
  --vision_codec_layers 4 \
  --vision_codec_train_steps 400 \
  --vision_codec_train_batch_size 2 \
  --vision_codec_train_lr 2e-4
```

### 19.3.3 训练参数选择

| 场景 | dim | tokens | batch_size | steps |
|------|-----|--------|------------|-------|
| **快速测试** | 256 | 512 | 4 | 50 |
| **标准训练** | 512 | 1024 | 2 | 400 |
| **高质量** | 1024 | 2048 | 1 | 800 |

## 19.9 结果分析

### 19.9.1 计算准确率

```python
import json

def evaluate_results(jsonl_path):
    with open(jsonl_path, 'r') as f:
        results = [json.loads(line) for line in f]

    correct = sum(1 for r in results if r.get('correct', False))
    total = len(results)
    acc = correct / total if total > 0 else 0

    print(f"Total: {total}, Correct: {correct}, Accuracy: {acc:.2%}")
    return acc

evaluate_results('preds_jsonl/merged_results/gsm8k_merged.jsonl')
```

### 19.9.2 分析错误

```python
def analyze_errors(jsonl_path):
    with open(jsonl_path, 'r') as f:
        results = [json.loads(line) for line in f]

    errors = [r for r in results if not r.get('correct', False)]

    print(f"Total errors: {len(errors)}")

    # 分析停止原因
    stop_reasons = {}
    for r in errors:
        reason = r.get('stop_reason', 'unknown')
        stop_reasons[reason] = stop_reasons.get(reason, 0) + 1

    print("Stop reasons:", stop_reasons)

    # 分析 unparsable
    unparsable = sum(1 for r in errors if r.get('extraction_source') == 'unparsable')
    print(f"Unparsable: {unparsable}")

evaluate_errors('preds_jsonl/merged_results/gsm8k_merged.jsonl')
```

## 19.10 常见问题排查

### 19.10.1 训练问题

| 问题 | 原因 | 解决 |
|------|------|------|
| OOM | Batch size 太大 | 减小 `train_batch_size` |
| Loss 为 NaN | 学习率太高 | 降低 `train_lr` |
| 编解码器不收敛 | 数据太少 | 增加锚点文本数量 |

### 19.10.2 合并问题

| 问题 | 原因 | 解决 |
|------|------|------|
| 架构不匹配 | 不同参数训练 | 使用相同参数 |
| 对齐后性能下降 | 锚点文本不足 | 增加锚点文本 |

### 19.10.3 运行问题

| 问题 | 原因 | 解决 |
|------|------|------|
| OOM | Batch size 太大 | 使用 `OOM_RETRY_LEVELS` |
| 模型加载失败 | 模型名错误 | 检查 HuggingFace 模型名 |
| 结果为空 | 分区失败 | 检查日志 |
